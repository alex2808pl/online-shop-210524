В проект работи в классе добавлен файл DB.png, в котором описана примерная структура БД online магазина.
Для каждой таблицы создайте Entity класс для отражения ее в ORM.
Содайте такж интерфейс репозитория для работы с каждой таблицей.
Переработайте наши Service классы для работы с БД через интерфейсы репозиторией,
а также заполните первоначальными значениями наши таблицы используя метод,
помеченный аннотацией @PostConstruct.
Переработайте метод GET контроллера для отображения всех строк из таблицы БД.
1) Users - пользователи
2) Products - товары
*3) Cart - корзина
*4) CartItems - товары в корзине
*5) Orders - заказы
*6) OrderItems - товары в заказе
*7) Favorites - избранные товары
*8) Categories - категории товаров.
